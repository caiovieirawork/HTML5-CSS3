Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag b de forma NÃO SEMÂNTICA (sem significado, somente sobre a forma)

Nessa frase agora, temos um termo em destaque usando a tag strong de forma SEMÂNTICA (com a forma que queremos[negrito] + um significado[um trecho com destaque, ou seja, "strong"])


Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag i de forma NÃO SEMÂNTCA

Nessa frase temos um termo em ênfase usando a tag em de forma SEMÂNTICA

Texto Grande e Pequeno

Nessa frase pdemos ver um texto grande usando a tag big de forma NÃO SEMÂNTICA

Mas nessa, podemos ver um texto pequeno usando a tag small de forma SEMÂNTICA (letras pequenas são uma forma mas não são consideradas obsoletas como a tag big, porque pode significar algo "Leia as letras pequenas nesse documento, pode ser levada então de forma semântica")

Texto Marcado

Aqui podemos ver um texto marcado igual um marca texto usando a tag mark

Texto Deletado e Inserido

Aqui podemos ver um texto deletado usando a tag del e aqui um inserido usando a tag ins

Palavras acima e abaixo

Em situações como esta x12+30 usamos a tag sup para definir como acima e em situações como esta H2O usamos a tag sub para definir como abaixo
